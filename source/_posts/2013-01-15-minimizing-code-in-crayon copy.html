--- layout: post title: Minimizing Code in Crayon tags: - Crayon status: publish type: post published: true meta: _edit_last: '1' dsq_thread_id: '1037676918' --- In the latest version of Crayon, you can choose to hide the code until the user expands it. This doesn't apply to inline code. You can enable/disable it for individual Crayons using the Tag Editor. If a title is set in the Tag Editor it is used in place of "Click To Expand Code".
