I've packaged the game in a single zip, all you need is to have the Java Runtime Environment installed on your computer and then run the .exe inside the zip. If you're on a Mac or Linux machine, you can run the .jar as long as you set the max memory heap to something like 128MB, otherwise the game dies after it fills the available memory. This wasn't a problem on my Mac, but I had to make custom exe's for Windows because the default memory heap was lower. In case the game doesn't start you can always set your heap larger (say min 32 to max 128MB) with:
java -Xms32m -Xmx128m -jar shadow_wing.jar
